

# ddm-cu

#### 介绍

代代米网站是一个现代化的Web项目，旨在为用户提供优质的内容和服务体验。该项目采用主流的前后端分离架构，支持响应式设计，能够在各种设备上提供良好的用户体验。

#### 软件架构

- **前端框架**: 主流HTML5/CSS3/JavaScript技术栈
- **响应式设计**: 支持PC端和移动端访问
- **模块化开发**: 采用模块化设计理念，便于维护和扩展

#### 安装教程

1. 确保本地环境已安装Node.js（建议v14.x及以上版本）
2. 克隆项目到本地：
   ```bash
   git clone https://gitee.com/caymark/ddm-cu.git
   ```
3. 进入项目目录并安装依赖：
   ```bash
   cd ddm-cu
   npm install
   ```

#### 使用说明

1. 开发模式运行：
   ```bash
   npm run dev
   ```
2. 生产环境构建：
   ```bash
   npm run build
   ```
3. 访问项目：在浏览器中打开 http://localhost:端口号
